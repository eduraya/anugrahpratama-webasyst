Класс общего назначения .flexbox переводит элемент в режим отображения display: flex;.

Никакого специфического для бекенда Webasyst оформления этот класс не навязывает, но будет полезен в качестве структурного контейнера, содержащего внутри себя разные интерфейсные компоненты: заголовки, меню, кнопки, текстовые поля, блок с произвольным контентом .box.

.flexbox.space-XX

С помощью дополнительных классов .flexbox.space-XX добавляются внутренние отступы для дочерних элементов, где XX — значение в пикселях с шагом 4 — 0, 4, 8, 12, 16, 20, 24, 28, 32, 36, 40, 44, 48. Шаг в 4 пикселя выбран исходя из соображений сохранения визуальной сетки интерфейса и практической полезности. Пиксели будут пересчитаны в rem.

.flexbox.fixed

В этом режиме ячейки равномерно заполняют свободное пространство по аналогии с таблицей. В отличие от .full-width, пространство является шириной ячейки.

Если контент ячейки не влезает с рассчитанное пространство, ячейка будет растянута. Такова особенность display: flex. Если нужно жестко зафиксировать ячейку, используется .tablebox вместо .flexbox.

{_render class="fixed"}
{_render|escape class="fixed"}

.flexbox.vertical

Вертикальное расположение элементов flexbox. Полезно при заполнении «столбцов» и меню типа сайдбара, где отдельные элементы могут занимать всю доступную высоту экрана и иметь внутреннюю прокрутку. Или же при изменении режима отображения блока «на лету» с помощью JavaScript.

.flexbox.vertical-mobile

Вертикальное расположение элементов flexbox будет включено только на мобильных устройствах (<=760px по ширине).

.flexbox.wrap

Flexbox будет «разорван», если дочерние элементы не помещаются по ширине в одну строку (flex-wrap: wrap;).

.flexbox.wrap-mobile

Аналогично, но только на мобильных устройствах (<=760px по ширине).
